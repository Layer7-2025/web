## Layer7 Web

### 선린인터넷고등학교 해킹 전공 동아리

---

### Installation

```sh
git clone https://github.com/Layer7-2025/web

pnpm install
```

### Development Start

```sh
pnpm run dev
```

### Production Start

```sh
docker compose up -d
```
